# roskeigan_motor_usb

## Installation
<p>rosのworkspaceに移動</p>  

    $ cd catkin_ws/src
    
<p>パッケージをインストール</p>

    $ git clone https://github.com/keigan-motor/roskeigan_motor_usb.git
    $ cd ..
    $ catkin_make

## Demo
<p>launchファイルから実行</p>

    $ roslaunch roskeigan_motor_usb velocity_control_keigan.launch
